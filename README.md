This repository serves as a personal archive and a "cheat sheet" for Linear Algebra. It is designed to distill complex mathematical concepts into concise, readable Markdown documents.

The content primarily follows the flow of **Gilbert Strang's "Introduction to Linear Algebra"** and his MIT OpenCourseWare lectures, enriched with additional insights for practical application in **Machine Learning** and **Data Science**.

* **Cheat Sheets:** Quick reference guides for formulas, theorems, and matrix properties.
